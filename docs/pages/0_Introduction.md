## Features

- HID device with USB and BLE connection  
- split keyboards time synchronization  
- keys signal debouncing  
- user configuration check  
- communication with messages  
- possible two way communication  
- unique functional keys  
- hardware VIM  

## Key Types

- standard  
- layers  
- hold tap  
- tap dance  
- sticky  
- media  
- keys sequence  
- vim
